Paypal

安装指南

  1. 登录Paypal，点击此处来创建一个Paypal捐赠按钮。

  2. 在“选择按钮样式”(Choose button style)页面选择“国家/地区”(Country/Region)和“语言”(Language)，点击“继续”(Continue) 进入下一页面。


  3. 在“添加机构详情”(Add organization details)页面中，选择“使用账号ID”(Use account ID)或“使用Email地址”(Use email address) 作为唯一账号标识符。 然后点击“继续”(Continue)进入下一页面。


  4. 在“设置捐赠数额”(Set donation amounts)页面选择“接收的货币种类”(Currency you’ll receive donations in)，捐赠数额选择 “任意数额”(Any amount)。 此捐赠按钮暂不支持“指定数额”的捐赠选项。 点击“结束并获取代码”(Finish and Get Code)按钮进入下一页面。


  5. 从“按钮HTML代码”(Button HTML)页面中复制business和currency_code两项的值。 将它们填写到主题配置的business和currency_code设置中。
